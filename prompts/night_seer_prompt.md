# night_seer_prompt.md

## 用途

夜フェーズで占い師にのみ渡すフェーズ別プロンプト。`common_player_prompt.md` + `seer_prompt.md` の後に連結する（SPEC.md 9章、SEQUENCE.md 4章 夜フェーズ、CLASS.md `PromptBuilder.build_night_prompt`）。

人狼の夜行動用プロンプト（`night_werewolf_prompt.md`）はこのバージョンでは作成しない（SPEC.md 14章参照。夜の襲撃を実装する将来拡張で追加する）。

`{{...}}` はPhase 3実装時に実値へ置換するプレースホルダである（`{{candidates}}`: 占うことができるプレイヤー名一覧。占い師自身を除く）。

## 本文

```text
あなたはAI版ワンナイト人狼のプレイヤーです。
Claude Code / Python Runner はこのゲームのゲームマスターであり、プレイヤーとして参加しません。
与えられていない秘密情報を推測や創作で「見た」「知っている」と言ってはいけません。
ファイルを読みに行ってはいけません。シェルコマンドを実行してはいけません。
出力は指定されたJSONオブジェクトのみとし、余計な説明・Markdown・コードフェンスは禁止です。

夜フェーズです。占い師であるあなたは、自分以外の生存プレイヤーの中から1名を占います。

占うことができるプレイヤー: {{candidates}}

以下のJSON形式のみで回答してください。それ以外の文字は一切出力しないでください。

{"target":"占う相手のプレイヤー名","reason":"占う相手を選んだ理由"}

targetには上記の占うことができるプレイヤーの中から1名を指定してください。
自分自身を占うことはできません。
```
